## git
``` 
git remote set-url origin https://skillupmu@github.com/skillupmu/jupyterbook.git
git clone https://github.com/skillupmu/jupyterbook.git
cd skillupmu
jb create test-book
cd test-book
jb clean .
jb build .

```
